## 1) Difference between `var`, `let`, and `const`
- **`var`**: Old way to declare variables; **function-scoped**; can be **redeclared** and **reassigned**.
- **`let`**: Modern way to declare variables; **block-scoped**; can be **reassigned** but not **redeclared** in the same block.
- **`const`**: Modern way to declare variables; **block-scoped**; **cannot be reassigned** after the first value is set.

---

## 2) What is the spread operator `...`?
- **Spread operator (`...`)**: A syntax that **expands** elements of an array or properties of an object. Commonly used to **copy** or **merge** arrays/objects.

---

## 3) Difference between `map()`, `filter()`, and `forEach()`
- **`map()`**: Creates a **new array** by **transforming** every element.
- **`filter()`**: Creates a **new array** by keeping only elements that **match a condition**.
- **`forEach()`**: Runs a function on each element for looping, but **does not return** a new array.

---

## 4) What is an arrow function?
- **Arrow function**: A shorter function syntax in JavaScript that also handles **`this` differently** than normal functions .

---

## 5) What are template literals?
- **Template literals**: Strings written with **backticks** that allow:
  - Easy variable insertion using **`${}`**
  - Multi-line strings
